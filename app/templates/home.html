Predict the Rarity (Rank) of a Hypothetical Monster

The purpose of this project is to predict the rarity of a hypothetical monster with a confidence level based on its stats such as level, health, energy, and sanity. The monster is randomly generated from a Python library. You can view the list of monsters by clicking the DATA tab. Visualization of the data with user interaction is displayed under the VIEW tab. Finally, the MODEL tab will show you the prediction of the rarity based on the numbers you enter in the fields. Have fun!

Tech Stack

  • Logic: Python3
  • API Framework: Flask
  • Templates: Jinja2
  • Structure: HTML5
  • Styling: CSS3
  • Database: MongoDB
  • Graphs: Altair
  • Machine Learning: Scikit
